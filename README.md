# Cricket Rating System
This is a cricket rating system developed for rating teams' batting, bowling, and fielding attributes out of 100. Currently, only IPL teams have been rated based on match data from the recently concluded season, ratings for international teams coming soon. I am still working on the project.
Following technologies were used:
1. React (With NextJS as the framework)
2. Flask
3. PostgreSQL.
