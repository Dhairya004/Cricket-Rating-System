# Cricket Rating System
This is a cricket rating system developed for rating teams' batting, bowling, and fielding attributes out of 100. Currently, only IPL teams have been rated based on match data from the recently concluded season, ratings for international teams are guesswork. Project still in work.
Following technologies were used to make this project:
1. React (With NextJS as the framework)
2. Flask
3. MS Excel for keeping track of match records.
